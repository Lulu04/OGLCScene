This is a project template for a game based on OGLCScene.

1) Copy the content of this folder (with the sub-folders) in your game folder
2) open 'project_oglcscene.lpi' with Lazarus
3) rename it  with Project->Save As-> the name of your project
4) delete all files starting with 'project_oglcscene'
5) open your new project with Lazarus, and in menu Project Options -> Compiler Options -> Path : correct the path to OGLCScene\source folder.
6) in unit "u_app.pas" modify the values of the constant SCREEN_WIDTH_AT_DESIGN_TIME and SCREEN_HEIGHT_AT_DESIGN_TIME to define the size of the screen when you design the graphics.
